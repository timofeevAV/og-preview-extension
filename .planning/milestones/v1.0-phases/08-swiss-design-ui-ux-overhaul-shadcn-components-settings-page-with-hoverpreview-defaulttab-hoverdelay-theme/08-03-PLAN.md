# Plan 08-03: Settings Page UI

## Goal
Install required shadcn components via the shadcn MCP/CLI, then build the `SettingsPage` component with 4 settings (hoverPreview, defaultTab, hoverDelay, theme). Wire it into `App.tsx` (replacing the placeholder from Plan 08-01). Apply Swiss Design principles to the settings layout.

## must_haves
- 6 shadcn components installed: `switch`, `select`, `slider`, `label`, `separator`, `button`
- `SettingsPage` component renders all 4 settings with proper shadcn components
- Settings read from `getSettings()` on mount; changes saved via `setSetting()`
- Gear icon button in popup to navigate to settings (`onOpenSettings` prop from App)
- Back button in settings header returns to main view
- Layout follows Swiss Design: 8px grid, `Separator` between sections, left-aligned labels

## Tasks

<task id="08-03-1">
### Install shadcn components via MCP

Use the shadcn MCP `get_add_command_for_items` and then run the CLI to install the required components. Install all at once:

```bash
pnpm dlx shadcn@latest add switch select slider label separator button
```

This creates:
- `components/ui/switch.tsx`
- `components/ui/select.tsx`
- `components/ui/slider.tsx`
- `components/ui/label.tsx`
- `components/ui/separator.tsx`
- `components/ui/button.tsx`

If any component is already present, the CLI will skip it. Run from the project root.
</task>

<task id="08-03-2">
### Create SettingsPage component

Create `entrypoints/popup/components/SettingsPage.tsx`:

```tsx
import { useEffect, useState } from 'react';
import { getSettings, setSetting, type OgPreviewSettings } from '@/lib/settings';
import { Switch } from '@/components/ui/switch';
import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from '@/components/ui/select';
import { Slider } from '@/components/ui/slider';
import { Label } from '@/components/ui/label';
import { Separator } from '@/components/ui/separator';
import { Button } from '@/components/ui/button';
import { HugeiconsIcon } from '@hugeicons/react';
import { ArrowLeft01Icon } from '@hugeicons/core-free-icons';

interface SettingsPageProps {
  onBack: () => void;
}

export function SettingsPage({ onBack }: SettingsPageProps) {
  const [settings, setSettings] = useState<OgPreviewSettings | null>(null);

  useEffect(() => {
    getSettings().then(setSettings);
  }, []);

  if (!settings) return null;

  async function update<K extends keyof OgPreviewSettings>(key: K, value: OgPreviewSettings[K]) {
    await setSetting(key, value);
    setSettings(prev => prev ? { ...prev, [key]: value } : prev);
  }

  return (
    <div className="w-[380px] flex flex-col">
      {/* Header */}
      <div className="flex items-center gap-2 px-4 h-10 border-b border-border">
        <Button variant="ghost" size="icon" onClick={onBack} className="h-7 w-7 -ml-1">
          <HugeiconsIcon icon={ArrowLeft01Icon} size={16} color="currentColor" />
        </Button>
        <span className="text-sm font-semibold tracking-wide">Settings</span>
      </div>

      {/* Settings list */}
      <div className="flex flex-col">

        {/* Hover Preview */}
        <div className="flex items-center justify-between px-4 py-3">
          <div className="flex flex-col gap-0.5">
            <Label htmlFor="hover-preview" className="text-sm font-medium cursor-pointer">
              Hover Preview
            </Label>
            <span className="text-xs text-muted-foreground">Show link preview on hover</span>
          </div>
          <Switch
            id="hover-preview"
            checked={settings.hoverPreview}
            onCheckedChange={(v) => update('hoverPreview', v)}
          />
        </div>

        <Separator />

        {/* Default Tab */}
        <div className="flex items-center justify-between px-4 py-3">
          <div className="flex flex-col gap-0.5">
            <Label className="text-sm font-medium">Default Tab</Label>
            <span className="text-xs text-muted-foreground">Tab shown on expand</span>
          </div>
          <Select
            value={settings.defaultTab}
            onValueChange={(v) => update('defaultTab', v as OgPreviewSettings['defaultTab'])}
          >
            <SelectTrigger className="w-32 h-8 text-xs">
              <SelectValue />
            </SelectTrigger>
            <SelectContent>
              <SelectItem value="previews">Previews</SelectItem>
              <SelectItem value="metadata">Metadata</SelectItem>
            </SelectContent>
          </Select>
        </div>

        <Separator />

        {/* Hover Delay */}
        <div className="flex flex-col gap-2 px-4 py-3">
          <div className="flex items-center justify-between">
            <div className="flex flex-col gap-0.5">
              <Label className="text-sm font-medium">Hover Delay</Label>
              <span className="text-xs text-muted-foreground">Wait before showing preview</span>
            </div>
            <span className="text-xs tabular-nums text-muted-foreground min-w-[52px] text-right">
              {settings.hoverDelay} ms
            </span>
          </div>
          <Slider
            value={[settings.hoverDelay]}
            onValueChange={([v]) => update('hoverDelay', v)}
            min={0}
            max={2000}
            step={50}
            className="w-full"
          />
        </div>

        <Separator />

        {/* Theme */}
        <div className="flex items-center justify-between px-4 py-3">
          <div className="flex flex-col gap-0.5">
            <Label className="text-sm font-medium">Theme</Label>
            <span className="text-xs text-muted-foreground">Appearance override</span>
          </div>
          <Select
            value={settings.theme}
            onValueChange={(v) => update('theme', v as OgPreviewSettings['theme'])}
          >
            <SelectTrigger className="w-32 h-8 text-xs">
              <SelectValue />
            </SelectTrigger>
            <SelectContent>
              <SelectItem value="system">System</SelectItem>
              <SelectItem value="light">Light</SelectItem>
              <SelectItem value="dark">Dark</SelectItem>
            </SelectContent>
          </Select>
        </div>

      </div>
    </div>
  );
}
```

**Theme change side-effect:** When `theme` changes, immediately apply it to the current popup document:
```ts
// Inside update() after setSetting, for 'theme' key:
if (key === 'theme') {
  const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
  const isDark = value === 'dark' || (value === 'system' && prefersDark);
  document.documentElement.classList.toggle('dark', isDark as boolean);
}
```

Add this inside the `update` function in `SettingsPage.tsx`.
</task>

### Add gear icon to CompactCard

Add an `onOpenSettings?: () => void` prop to `CompactCard`. Add a gear icon button in the top-right corner of the title/info area (or as an absolute-positioned button over the image area top-right):

1. Import `HugeiconsIcon` from `@hugeicons/react` (already used) and `Settings01Icon` from `@hugeicons/core-free-icons`
2. Add `onOpenSettings?: () => void` to `CompactCardProps`
3. Render the gear button as a small ghost icon button positioned top-right within the image banner:

```tsx
{/* Image banner with gear overlay */}
<div className="relative w-full overflow-hidden" style={{ aspectRatio: '1.91' }}>
  {image ? (
    <img src={image} alt={...} className="w-full h-full object-cover" />
  ) : (
    <div className="w-full h-full bg-muted flex items-center justify-center">
      <span className="text-xs text-muted-foreground">No image</span>
    </div>
  )}
  {onOpenSettings && (
    <button
      onClick={onOpenSettings}
      className="absolute top-2 right-2 w-7 h-7 rounded flex items-center justify-center bg-black/30 hover:bg-black/50 text-white transition-colors"
      aria-label="Open settings"
    >
      <HugeiconsIcon icon={Settings01Icon} size={14} color="currentColor" />
    </button>
  )}
</div>
```

4. Pass `onOpenSettings` from `App.tsx` to `CompactCard`:
```tsx
<CompactCard
  ogData={ogData as OgData}
  isPartial={isPartial}
  expanded={expanded}
  onToggleExpand={() => setExpanded(v => !v)}
  onOpenSettings={() => setView('settings')}
/>
```

Also update `EmptyState` and loading/error states to show the gear icon — add a minimal header with gear icon for those states so users can access settings from any state.
</task>
